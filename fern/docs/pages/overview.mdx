## Quickstart

Your team can get started with the ProjectManager API right away - here's how.

1. All ProjectManager.com accounts have access to the API. [Create a free trial now!](https://developer.projectmanager.com/reference/www.projectmanager.com/freetrial?edition=d)

2. Once you have signed up for an account, [create an API key](https://developer.projectmanager.com/reference/api-keys) within ProjectManager. Remember to save your API key in your company's secure credential storage system!

3. Download the appropriate software development kit for your favorite programming language, or take a look at our example programs:

## Using the API

The ProjectManager.com API allows you to synchronize data from other systems to your project files and data. 

<Callout intent="info">
The following are ideas for integrating with the ProjectManager API.
</Callout>

<br />

- Notify your accounting system when a task is completed, so they can send an invoice to the customer
- Import data from your internal IT systems into tasks
- Run a security scan on your software and automatically create a task for each risk
- Automatically synchronize comments to tasks
