The ProjectManager Legacy API v1 is only accessible to customers on custom contracts.  

### Upgrading from Legacy API v1 to API v4

To begin using REST API v4, you must upgrade your account.  This service is complimentary and can be scheduled by contacting your customer success representative at support@projectmanager.com.
